# 分享功能方案文档

## 概述

在 SWUFE 社团活动资讯平台添加分享功能：用户点击分享按钮，弹出二维码卡片，扫码可直接访问平台。

---

## 方案一：客户端 QR 码生成（推荐）

### 技术栈

| 组件 | 选择 | 理由 |
|------|------|------|
| QR 码库 | [qrcodejs](https://github.com/davidshimjs/qrcodejs) (纯前端) | 无依赖、轻量 (约 15KB gzipped)、中国 CDN 可用 |
| 承载方式 | 独立弹窗卡片 | 复用现有 dialog 样式体系，视觉一致 |

### 数据传输示意

```
用户点击分享 → JS 获取当前 URL → qrcodejs 在 canvas 上绘制二维码
                                         ↓
                                  弹窗显示二维码卡片 (含简介文案)
                                         ↓
                                  用户长按/截图 → 分享到微信/QQ
```

### 实现要点

1. **QR 码库加载**
   - 从 `cdnjs` 加载 `qrcodejs`：中国节点未封锁，可正常访问
   - 备用方案：将库内联到 `js/` 目录，完全避免外部依赖

2. **分享按钮位置**
   - 方案 A：`<header>` 右侧，搜索框旁边 —— 显眼，符合用户习惯
   - 方案 B：浮动按钮，固定于页面右下角 —— 不干扰布局，但可能被忽略
   - → 推荐方案 A，与搜索框并列，图标为 `📤` 或 SVG 分享图标

3. **二维码卡片内容**
   - 平台名称："SWUFE 社团活动资讯"
   - 简短介绍："聚合西南财经大学社团活动资讯"
   - 二维码：指向 `window.location.origin`
   - 底部提示："扫码访问社团资讯平台"

4. **交互流程**
   ```
   点击分享按钮 → 打开半透明遮罩 → 显示二维码卡片
                                      ↓
                     卡片包含：Logo + 简介 + QR码 + 提示文字
                                      ↓
                   点击遮罩 / 关闭按钮 → 关闭卡片
   ```

5. **样式**
   - 复用现有 `.dialog-overlay` / `.dialog-panel` 结构
   - 卡片宽度限制 320px，居中
   - QR 码 200×200px，居中显示
   - 深色/浅色模式自适应

### 优缺点

| 优势 | 劣势 |
|------|------|
| 纯前端，无需后端 | 需要加载外部 JS 库 |
| QR 码实时生成，动态 URL 也支持 | - |
| CDN 在中国可访问 | - |

## 方案二：API 生成 QR 码（简单但不可靠）

### 思路

直接使用 `https://api.qrserver.com/v1/create-qr-code/` 生成 QR 码图片：

```html
<img src="https://api.qrserver.com/v1/create-qr-code/?size=200x200&data=https://..." alt="二维码">
```

### 优缺点

| 优势 | 劣势 |
|------|------|
| 实现极简，无需 JS 库 | **API 在中国被墙，不可用** |
| - | 生成速度依赖网络 |
| - | 无法离线使用 |

→ **否决**：与中国可访问目标冲突。

---

## 方案三：纯手写 QR 码生成器内联（无需外部加载）

### 思路

将微型 QR 码生成算法（如 [qr.js](https://github.com/cozmo/qr.js) ~5KB）直接打包到 `app.js` 中，零外部请求。

### 优缺点

| 优势 | 劣势 |
|------|------|
| 零外部依赖，加载最快 | 实现稍复杂（需打包算法） |
| 完全离线可用 | 维护性略差 |
| 不被任何网络封锁影响 | - |

---

## 推荐方案

**方案一（qrcodejs）**，理由：
1. 成熟稳定，社区广泛使用
2. CDN 在中国可访问，加载可靠
3. 开发成本低，2-3 行代码即可生成
4. 万一 CDN 不可用，可快速切换为内联

## 改动文件清单

| 文件 | 改动内容 |
|------|---------|
| `site/index.html` | 添加分享按钮（header 搜索框旁）、添加 qrcodejs CDN 引用、添加二维码卡片 HTML 结构 |
| `site/css/style.css` | 添加二维码卡片样式（约 40 行） |
| `site/js/app.js` | 添加分享按钮事件、二维码生成逻辑（约 30 行） |

## 预期效果

1. 用户点击分享按钮 → 弹窗显示二维码卡片
2. 卡片包含：平台名称 + 简介 + 200×200 QR 码 + 扫码提示
3. 用户长按二维码 → 保存/分享到微信/QQ
4. 好友扫码 → 直接打开社团活动平台
